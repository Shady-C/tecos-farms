# Teco's Farms — Documentation Index

**Purpose:** Single entry point for all project documentation. Use this index to find the right doc for your role or task.

**Abstract:** This documentation set covers the Teco's Farms Order Management System (OMS)—a lean, mobile-first web app for Tanzania that replaces WhatsApp-to-spreadsheet ordering. It includes project overview, requirements, architecture, API reference, codebase map, technical specs, process, testing, deployment, user/support guides, and standards.

**Audience:** Developers, QA, product managers, and new team members.

**Document version:** Last updated 2026-02-24; aligns with Phase 1 MVP.
